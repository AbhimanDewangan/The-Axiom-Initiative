# Axiom Initiative — Jaw‑dropping landing page (no backend)

A premium-feel, extremely reactive landing page template built with:
- React + Vite
- Tailwind CSS
- Framer Motion
- Lightweight canvas particles (no extra deps)
- Code snippet showcase with copy button

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Customize

- Site title/meta: `index.html`
- Main content: `src/App.jsx`
- Palette/gradients: `src/styles.css`
- Favicon: `public/favicon.svg`

## Deploy

This is a static build. Deploy the `dist/` folder to:
- GitHub Pages
- Netlify
- Vercel
- Any static host
